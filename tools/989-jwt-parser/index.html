                About JWT Tokens
            

            
    
                
  • JWT = JSON Web Token for secure transmission
    • 
                
  • Three parts: Header, Payload, Signature
    • 
                
  • Base64URL encoded (not encrypted)
    • 
                
  • Standard claims: iat, exp, sub, iss, aud
    • 
                
  • Supports HS256, RS256, ES256 algorithms
    • 
                
  • 100% local processing
